Library
Class

HttpClient
Name

POST
Description Performs a POST request, sending the specified parameters as data in the same way a web browser submits a form.
Syntax 
client.POST(path, params)
client.POST(path, params, files)
Parameters
client HttpClient: any variable of type HttpClient
path String: an absolute path to a file or script on the server
params HashMap: a collection of parameters to send to the server
files HashMap: a collection of files to send to the server
Returns HttpRequest
Usage Application
